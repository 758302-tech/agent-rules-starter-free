# Free Sample — Agent Rules Starter Pack

Give your AI coding agent a memory layer before it edits your repo.

Copy these files into a project root to give AI coding agents a basic operating layer:

- `AGENTS.md`
- `CLAUDE.md`
- `.cursor/rules/agent-baseline.mdc`

## Install

From this repository:

```bash
cp AGENTS.md /path/to/your-project/AGENTS.md
cp CLAUDE.md /path/to/your-project/CLAUDE.md
mkdir -p /path/to/your-project/.cursor/rules
cp .cursor/rules/agent-baseline.mdc /path/to/your-project/.cursor/rules/agent-baseline.mdc
```
